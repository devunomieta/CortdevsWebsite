A sophisticated, high-performance agency website and management portal built with React, Supabase, and Tailwind CSS. This platform features a premium "tech-vibe" public interface, an advanced administrative CRM, and a secure client portal.

## 🚀 Quick Start

1. **Install Dependencies**: `npm install`
2. **Environment Variables**: Configure `.env` based on `docs/setup.md`.
3. **Database Setup**: Execute scripts in Supabase (see `docs/setup.md#database-migration`).
4. **Run Locally**: `npm run dev`
